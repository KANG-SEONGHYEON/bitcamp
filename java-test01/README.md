# 연습 프로젝트

## 요구사항
 
- 도서 정보 등록
  - 도서명, 출판사, 출판일, 가격, 설명, 사진
  
- 도서 정보 목록 출력
  - 도서사진, 도서명, 출판사, 가격

- 도서 상세 정보 출력
  - 도서번호, 도서명, 출판사, 출판일, 가격, 설명, 사진 

- 도서 정보 변경
  - 도서명, 출판사, 출판일, 가격, 설명, 사진 

- 도서 정보 삭제
